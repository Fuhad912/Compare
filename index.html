About Comparé

Comparé is a focused football comparison experience designed to place two players side by side across a selected season. The goal is fast, clear decision support for fans, analysts, and curious observers.

Tech stack: semantic HTML for structure, modern CSS for presentation, vanilla JavaScript for interaction, and an api/ directory prepared for Vercel serverless functions.
